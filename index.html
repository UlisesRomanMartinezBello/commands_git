Lista de Comandos en Git

ID de comando Comando Función
1 git --version Nos muestra la version de git que tenosmo instalada.
2 git help Nos muestra información de ayuda sobre los comandos de git.
3 git help command Nos muestra información de ayuda sobre un comando en concreto.
4 git config --global user.name "username" Asocia un nombre de usuario con tu terminal de git que se usuará en los repositorios locales.
5 git config --global user.email "user@email.com" Asocia un correo electrónico con tu terminal de git que se usuará en los repositorios locales.
6 git init Inicializa un repositorio en git.
7 git status Nos muestra información sobre los commits y las ramas.
8 git add file/folder Añade los archivos o carpetas con los cambios al stage y los prepara para un commit.
9 git reset file/folder Quita del stage los archivos o carpetas que indiquemos.
10 git commit -m "message" Registra los cambios en el repositorio, debe indicar un texto descriptivo de las modificaciones que se realizan.
11 git checkout -- . Reecontruye el repositorio a como estaba en el último commit.
